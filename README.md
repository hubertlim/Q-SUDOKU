## Features

🎯 **Smart solver** — bitmask constraints + MRV heuristic for instant solving

🎲 **Unique puzzles** — generated on the fly with guaranteed single solution

🎓 **Learn mode** — 12 progressive challenges from 4×4 mini grids to hard 9×9

⚡ **Speed mode** — tap a number, then tap cells to rapid-fire place it

🎨 **4 themes** — Midnight, Nord, Rosé Pine, Paper (saved across sessions)

� **Auto-save** — resumes your game exactly where you left off

📱 **Responsive** — fits any screen without scrolling

🐳 **Dockerized** — one command to run locally

## How it works

The engine uses backtracking with bitmask constraint tracking. Each row, column, and 3×3 box stores used digits as a 9-bit integer, making candidate checks O(1) via bitwise ops. The MRV (Minimum Remaining Values) heuristic always picks the most constrained cell next, keeping the search tree small.

Puzzles are generated by filling a complete grid with randomized backtracking, then removing clues one at a time while verifying the solution stays unique.

| Easy       | 38    | 1000  |
| Medium     | 30    | 2000  |
| Hard       | 24    | 3000  |

